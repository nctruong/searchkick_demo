# Searchkick example

[searchkick][1] example for [autocomplete][2], [conversions][3] and [similar][4].

  [2]: https://github.com/ankane/searchkick#autocomplete

## Use

````ruby
bundle install
bundle exec rake db:migrate
bundle exec rake db:seed
rails server
visit locolhost:3000
````
